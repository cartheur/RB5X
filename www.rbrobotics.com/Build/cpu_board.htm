 CPU Diagram 5

 

(   )  Using antistatic devices, insert the following IC chips into the correct positions (notch faces towards top of board):

(   )  U1 is Decoder SN74LS139.

(   )  U5 is Driver MC1413.

(   )  U6 is Inverter SN74LS14.

(   )  U7 is Dual AND SN74LS21.

(   )  U9 is Voltage Inverter ICL7660.

(   )  U11 is Comparator LM392.

(   )  U12 is Voltage Sensor ICL8211.

(   )  U13 is Hex Buffer CD4503.

(   )  U15 is Decoder 74LS154.

(   )  U16, U17, U18, and U19 are Memory 2Kx8  HM6116.

(   )  U24 is Quad NAND 74LS00 

(   )  U4, U20, and U23 are Programmable I/O INS8255.

(   )  U8 is the Microprocessor INS8703.

(   )  Install the two board tabs. Use a pair of pliers to squeeze the retaining bars in.

This concludes your CPU Board assembly.  Set it aside until called for later.
